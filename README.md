# Employee Management System

## Project Overview
Employee Management System is a full-stack web application developed using Java Spring Boot and MySQL. The application helps manage employee records efficiently through a simple and user-friendly interface.

## Features
- Add Employee
- View Employee Records
- Update Employee Information
- Delete Employee Records
- REST API Integration
- Database Connectivity with MySQL

## Technologies Used
### Backend
- Java
- Spring Boot
- Spring Data JPA
- REST APIs

### Frontend
- HTML
- CSS
- JavaScript

### Database
- MySQL

## Project Architecture
The project follows a layered architecture:
- Controller Layer
- Service Layer
- Repository Layer
- Model Layer

## Functionalities
- Employee data management
- CRUD operations
- Frontend and backend integration
- Database interaction using JPA

## Learning Outcome
This project helped improve understanding of:
- Java Spring Boot
- REST API Development
- CRUD Operations
- Database Integration
- Frontend-Backend Communication
